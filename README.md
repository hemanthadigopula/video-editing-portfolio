## üöÄ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Sublime Text, etc.)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/hemanthadigopula/video-editing-portfolio.git
```

2. Navigate to the project directory:
```bash
cd video-editing-portfolio
```

3. Open `index.html` in your web browser:
   - Double-click the file, or
   - Right-click and select "Open with" your preferred browser, or
   - Use a local development server (e.g., Live Server in VS Code)

## ‚úèÔ∏è Customization

### Update Personal Information

1. **Edit `index.html`**:
   - Replace "HEMANTH" with your name
   - Update the hero section text
   - Add your showreel video URL (replace `YOUR_VIDEO_ID`)
   - Update the About section with your bio
   - Replace social media links in the Contact section

2. **Update Project Cards**:
   - Replace placeholder images with your project thumbnails
   - Update project titles and descriptions
   - Add links to your video projects

3. **Customize Colors** (in `style.css`):
   - Modify CSS variables in the `:root` section
   - Change gradient colors to match your brand

### Add Your Own Videos

1. **Showreel**:
   - Upload your showreel to YouTube
   - Get the video ID from the URL
   - Replace `YOUR_VIDEO_ID` in the iframe src

2. **Project Thumbnails**:
   - Create thumbnail images (recommended: 400x200px)
   - Upload to an `assets/images/` folder
   - Update image src in HTML
